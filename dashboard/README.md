# EventCore Dashboard

The tenant-scoped web dashboard (Next.js) lives here. It is not built yet —
scope, screens, and auth model are being defined in
[milestone M11](https://github.com/sharanggupta/eventcore/milestone/12),
starting with a DISCUSS wave (issue #39).

Grounding: the competitor capability matrix and the two unclaimed UI
opportunities (pull-consumer fleet/lag view, tenant-visible retention budget)
in [docs/product/dashboard-and-billing-research.md](../docs/product/dashboard-and-billing-research.md).
Design reference: Next.js 15 + React 19 + Tailwind v4 + TypeScript with a
custom glassmorphic component library.
